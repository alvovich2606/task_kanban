# Kanban Board

Простое веб-приложение для управления задачами по методологии Kanban.

## Возможности

- Создание задач с названием и описанием
- Перемещение задач между колонками: To Do → In Progress → Done и обратно
- Удаление задач
- Ограничение количества задач в работе (лимит In Progress)
- При уменьшении лимита лишние задачи автоматически возвращаются в To Do
- Цветные метки с произвольным текстом (до 5 на задачу)
- Повторное использование ранее созданных меток
- Скрытие и раскрытие описания задачи
- Сортировка задач по времени последнего перемещения
